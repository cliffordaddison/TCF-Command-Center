# TCF Canada B2+ Exam Prep Command Center

The **TCF Canada Command Center** is a self-directed language learning operating system designed to guide candidates through the rigorous preparation required for a B2+ score. It is built on the **4-3-2-1 Rhythm Algorithm**, ensuring a balanced approach to input, production, and verification.

## Core Philosophy: The 4-3-2-1 Rhythm
This system eliminates decision fatigue by automating your study schedule:
- **4 Days of Input/Analysis:** Deep dives into *A Moi Paris* and linguistic analysis.
- **3 Days of Recall/Production:** Active testing and creative production.
- **2 Days of Verification:** Integrated scoring and progress audits.
- **1 Day of Maintenance:** Light immersion to prevent burnout.

## Key Features
- **Mission Control:** A live session interface with active timers and pause protection.
- **Kanban Command:** Drag-and-drop task management from Backlog to Archive.
- **Skill Radar:** Real-time visualization of your Listening, Reading, Speaking, and Writing proficiency.
- **The 100% Rule:** Automated rescheduling logic for failed tests to ensure mastery.
- **Streak Protection:** Visual motivation to maintain daily consistency.
- **Exam Readiness Score:** An algorithmic calculation of your probability of hitting B2+ based on performance logs.

## Technical Stack
- **Frontend:** React with TypeScript and Tailwind CSS.
- **UI Components:** shadcn/ui and Framer Motion.
- **Backend:** Firebase (Firestore & Authentication).
- **State Management:** Real-time Firestore sync.
- **Scheduling:** Custom rhythm logic based on user start dates.

## Goal
The ultimate objective of this Command Center is to transform a 3-6 month preparation period into a predictable, data-driven journey. By the end of the training period, the user should possess not just the linguistic skills, but the tactical confidence to excel in the TCF Canada exam.
